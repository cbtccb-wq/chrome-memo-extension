# Quick Memo — Chrome拡張機能

新しいタブを自由配置のメモキャンバスに置き換える Chrome 拡張機能です。テキストと画像をドラッグ・リサイズ・ズーム・パンしながら、自動保存。

## 機能の概要

- **新しいタブを開くだけでメモ画面** — `chrome_url_overrides.newtab` で常駐
- **自由配置キャンバス** — ダブルクリックでテキスト追加、Ctrl+V またはドロップで画像挿入
- **ドラッグ / 8 方向リサイズ** — Shift で比率固定、Shift+ドラッグでグリッドスナップ (24px)
- **複数選択 / 矩形選択** — Shift+クリックでトグル追加、空白ドラッグで矩形範囲を選択
- **Ctrl+ホイールでズーム** — マウス位置を中心に拡大縮小、Space+ドラッグまたは中ボタンドラッグでパン
- **Undo / Redo** — Ctrl+Z / Ctrl+Shift+Z、ツールバーボタンでも操作可
- **コピー & ペースト** — Ctrl+C / Ctrl+V で要素を複製、画像は右クリックでクリップボードへ
- **エクスポート** — テキストのみの `.txt` と、画像込みの単独 `.html` の 2 種類
- **自動保存** — IndexedDB に 600ms デバウンスで保存、再起動後も復元
- **ダークモード対応** — OS の設定に自動追従

## 詳しい使い方

### テキスト要素

- **作成**: キャンバスの空白部分を **ダブルクリック** すると、その位置にテキストボックスが現れて即編集モードに入ります。
- **編集**: 既存のテキスト要素をダブルクリックすると `contentEditable` で編集モードへ。`Esc` で確定。
- **改行**: 編集中は通常の `Enter` で改行できます。

### 画像要素

- **追加**: 画像ファイルを **キャンバスにドラッグ&ドロップ**、もしくは画像をクリップボードにコピーした状態で **Ctrl+V**。
- **サイズ制限**: 1 枚あたり **5 MB** まで。超過分はステータスバーに警告が出ます。
- **自動縮小**: 巨大画像は表示中ビューポートの 45%×70% 以内に収まるよう自動で縮小されます。
- **画像コピー**: 画像要素を **右クリック** すると即座に PNG としてシステムクリップボードに書き込まれます（メニューは出ません）。テキスト要素や空白部分の右クリックはブラウザ既定動作のままです。

### 選択モデル

- **単独選択**: 要素をクリック。
- **トグル追加**: `Shift+クリック` で複数選択 / 解除。
- **矩形選択 (マーキー)**: 空白部分を **ドラッグ** すると半透明の選択矩形が表示され、リリース時に交差した要素がまとめて選択されます。`Shift+ドラッグ` だと既存選択に追加します。
- **プライマリ要素**: 最後に追加された 1 つが「プライマリ」になり、リサイズハンドル (8 方向の白い丸) はこの要素にだけ表示されます。複数選択中はリサイズ不可です。
- **全解除**: 空白を 1 回クリック（3px 未満の動き）。

### ドラッグとリサイズ

- **ドラッグ**: 選択中の **全要素** が同じ delta だけ移動します。最小座標が 0 を割らないよう全体でクランプされます。
- **グリッドスナップ**: ドラッグ中に `Shift` を押し続けると、**24px グリッド** にスナップします（背景のドットグリッドと同じピッチ）。
- **リサイズ**: プライマリ要素のハンドルをドラッグ。`Shift` を押し続けると **コーナーハンドルで縦横比を固定**できます。
- **微調整**: 矢印キーで 1px 単位、`Shift+矢印` で 10px 単位の移動（選択中の全要素に適用）。

### ズームとパン

- **ズーム**: `Ctrl+マウスホイール` で **マウス位置を中心に** 0.25 倍 〜 4 倍までズーム。
- **パン**: `Space を押しながらドラッグ`、もしくは **中ボタンドラッグ**。
- **リセット**: ステータスバーの **ズーム率ピル** (`100%` 等) をクリックすると、ズームとパンの両方が一発で初期位置に戻ります。
- **永続化なし**: 拡大率とパン位置はリロードすると 100% / 原点に戻ります（要素の位置は保持されます）。

### Undo / Redo

- **戻る**: `Ctrl+Z` または ツールバーの「戻る」ボタン。
- **進む**: `Ctrl+Shift+Z` または `Ctrl+Y`、ツールバーの「進む」ボタン。
- **履歴サイズ**: 最大 50 ステップ。古い履歴から自動的に破棄されます。
- **保持されるタイミング**: 要素追加 / 削除 / ドラッグ完了 / リサイズ完了 / 編集完了 / クリア / 矢印キー操作（500ms デバウンス）。
- **テキスト編集中**: `contentEditable` のネイティブ undo に処理を委譲（編集確定後の差分のみ履歴に積まれます）。

### コピー & ペースト

- **要素のコピー**: 選択中に `Ctrl+C` で内部クリップボードへ保存（複数選択していれば全部）。
- **要素のペースト**: `Ctrl+V` で元の位置から `+20px` ずらして複製、複製した要素群が新しい選択になります。
- **優先順位**: ペーストは **システムクリップボードの画像 > 内部要素クリップボード** の順で評価。画像が入っていれば画像が、無ければ内部要素が貼られます。
- **テキスト編集中**: どちらも素通しでブラウザのネイティブ動作になります。

### エクスポート

- **TXT**: テキスト要素のみを `\n\n---\n\n` で連結した素の `.txt` ファイル。
- **HTML**: 全要素を絶対配置でレイアウトした単独 `.html` ファイル。画像は data URL を直接埋め込むので、ファイル単体で完結します。
- **ファイル名**: `memo_YYYYMMDD_HHmm.{txt,html}` の形式で自動命名。

### 自動保存

- 配置や編集の **変更が起きるたびに 600ms デバウンス** で IndexedDB へ書き込みます。
- フッターのステータスピルが `保存中…` → `保存済み` と推移するので、保存状況を視覚的に確認できます。
- `alert()` は使わず、エラーや一時メッセージはすべてステータスピルに `flash()` で 2 秒間表示します。

### ダークモード

- OS の設定 (`prefers-color-scheme: dark`) に自動追従。アプリ内に切替トグルはありません。

| Shift+コーナーリサイズ | 縦横比を固定 |
| 矢印キー | 1px 移動 (Shift で 10px) |
| Delete / Backspace | 選択中の要素を削除 |
| Ctrl+ホイール | マウス位置を中心にズーム |
| Space+ドラッグ / 中ボタンドラッグ | パン |
| ズーム率ピルをクリック | ビューを 100% / 原点にリセット |

| 画像を右クリック | 画像を PNG でクリップボードにコピー |

## Chromeへのインストール方法

1. Chrome のアドレスバーに `chrome://extensions` と入力して開く
2. 右上の **「デベロッパーモード」** をオンにする
3. **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリック
4. このフォルダ（`chrome-memo-extension`）を選択する
5. 新しいタブを開くと Quick Memo のキャンバスが表示されます

## ファイル構成

```
chrome-memo-extension/
├── manifest.json       # 拡張機能の設定（Manifest V3）
├── newtab.html         # キャンバス画面の HTML
├── newtab.css          # スタイル（ダークモード対応）
├── newtab.js           # 配置・編集・自動保存などのロジック
├── generate-icons.js   # アイコン生成スクリプト（開発用）
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## アイコンを再生成したい場合

```bash
node generate-icons.js
```

外部パッケージ不要（Node.js 標準の `zlib` のみ使用）。
